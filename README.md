# 📘 Algorithm_Study

알고리즘 스터디에 오신 것을 환영합니다! 🎉 본 스터디는 다음과 같은 규칙을 가지고 진행됩니다.
<br/><br/>
## 📜 스터디 규칙

1. **하루에 하나의 알고리즘 풀기** 🧩
   - 매일 새로운 알고리즘 문제를 풀어보며, 자신의 코딩 실력을 향상시킵니다.
2. **오후 8시까지 PR 작성하기** 📝
   - 풀이한 문제에 대한 PR(Pull Request)를 매일 오후 8시까지 작성해주세요.
3. **일주일에 한 번 푼 알고리즘에 대해 의견 공유시간 갖기** 💬
   - 일주일에 한 번, 스터디원들과 함께 모여 그 주에 푼 문제들에 대한 의견을 공유합니다. 서로의 접근 방식과 해결 전략을 공유하며, 더 나은 풀이 방법을 함께 찾아봅니다.
<br/>

## 📚 문제 선정

- **너무 쉽지 않은 30분 정도의 풀이시간을 갖는 문제로 선정하기** 📅
  - 문제 선정 시, 대부분의 스터디원이 약 30분 동안 고민하며 풀 수 있는 난이도를 기준으로 합니다.
